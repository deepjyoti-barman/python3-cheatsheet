## Notes on `01_hello_world.py`
- `print()` - This is a built-in Python function that displays output to the console/terminal. 
- `print()` takes whatever you pass to it and shows it on the screen.
- Syntax: `print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)`
- `"hello, world!"` - This is a string (text data) enclosed in double quotes.
- In Python, you can use either double quotes (`"`) or single quotes (`'`) to define strings.
- Command to execute: `python <file-name>.py`
